Research & References

Cognitive Diagnostic Assessment

NeuroQuiz implements Cognitive Diagnostic Assessment (CDA) principles, particularly the Rule Space Method, to provide detailed insights into knowledge mastery and learning patterns beyond aggregate scores.

Rule Space Method

Reference: Tatsuoka, K. K. (2009). Cognitive assessment: An introduction to the Rule Space Method. Routledge.

This foundational work introduces the Rule Space Method (RSM), a cognitive diagnostic technique that transforms item response patterns into measurable attribute mastery probabilities. RSM helps interpret test results beyond aggregate scores by identifying underlying knowledge strengths and weaknesses, enabling customized assessment feedback. It has been applied in large-scale assessments such as the PSAT and other educational diagnostics.

Executive Function Indicators

Our analytics incorporate educational executive function indicators including processing speed, impulsivity control, analytical thinking, cognitive endurance, and self-regulation to provide comprehensive cognitive profiling.

Adaptive Learning

Based on cognitive load theory and personalized learning algorithms that adjust content difficulty in real-time through the RB-ADA algorithm.
